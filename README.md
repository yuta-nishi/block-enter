ChatGPTのページでEnterによる送信を防ぐ拡張機能。\
[Chrome Web Storeはこちら](https://chrome.google.com/webstore/detail/block-enter/epcbggfjoekpdmenggjelljjegjdbpog)

## 使い方
Enter：改行 \
Ctrl/Cmd + Enter：送信

拡張機能ボタンを押すことでon/off切り替えが可能。
